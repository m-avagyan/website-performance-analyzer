### Getting Started

Installation

-   `git clone https://github.com/m-avagyan/website-performance-analyzer.git`
-   `cd website-performance-analyzer`
-   `nvm use`
-   `npm install`

Execution

-   `npm run analyze -- -u <URL> -c <CATEGORIES> -f <FORMAT>` - generating a new report for your website
